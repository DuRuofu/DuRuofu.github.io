HTML+CSS笔记 (10.颜色单位)

本文最后更新于：7 个月前

颜色单位

在css里可以直接使用颜色来设置各种颜色(不方便）

RGB值

RGB通过三种颜色的不同浓度来调配出不同颜色
R：red
G:green
B:blue
每一种颜色的范围都在0-255之间
语法：RGB（红色，绿色，蓝色）
这里是光的三原色：所以都为0时为黑色，都为255为白色

RGBA

A:在RGB基础上多一个不透明效果（1不透明——0完全透明）

十六进制的RGB值

语法：#红色绿色蓝色
颜色浓度：00-ff
如果符号两位两位相同，可以简写：#aabbcc –>#abc

HSL值和HSLA值

H：色相（0~360）
S：饱和度 颜色浓度（0%——100%）
L：亮度 颜色的亮度（0%——100%）
A:不透明度
